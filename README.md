BOTus
===================

[![Build Status](https://travis-ci.org/MABOOTYNG-RACKY/BOTus.svg?branch=master)](https://travis-ci.org/MABOOTYNG-RACKY/BOTus)

## Setup
- Copy the `.env.example` file and create a new `.env` file
- Run these
```sh
npm install
npm run dev
```

## Build
```sh
npm run build
```
